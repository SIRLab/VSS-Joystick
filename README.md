Mais informações podem ser encontradas em [VSS-SDK][vss-sdk].

Installation
------------
[INSTALL][install]

Cloning
-------
```
$ git clone --recursive https://github.com/SIRLab/VSS-Joystick.git
```

Development and Use
-------------------

```
$ make proto
$ make 
$ make run
```
